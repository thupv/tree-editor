# tree-editor

Very simple demo to build a hierarchy editor with undo/redo feature

## Demo
https://tree-editor-thupv.vercel.app/

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
